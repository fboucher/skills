# Agent Skills 

These are personal agent skills to help me when using AI. It's a work in progress, and even changing repo.

To add a skill use the following command:

```bash
npx skills add fboucher/skills/grill-me
```

- **consistent-lexicon** — Extract and formalize domain terminology from the current conversation into a consistent glossary, saved to `LEXICON.md`.

- **edit-video-blog** — Write a polished technical blog post from a video transcript and an initial draft.

- **grill-me** — Get relentlessly interviewed about a plan or design until every branch of the decision tree is resolved.

- **improve-codebase** — Explore a codebase to find architectural friction, surface refactoring opportunities, and propose module-deepening RFCs as GitHub issues.

- **prd-to-issues** — Break a PRD into independently-grabbable GitHub issues.

- **tdd** — Build features or fix bugs using test-driven development with a strict red-green-refactor loop.

- **write-a-prd** — Create a Product Requirement Document (PRD) through an interactive interview, codebase exploration, and module design. Filed as a GitHub issue.

- **write-a-skill** — Create new agent skills with proper structure, progressive disclosure, and bundled resources.
